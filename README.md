## Dockerize

Run the `docker.sh` file, it will create a new container image with the tag `jhordyess/simloc` and run it; so you can visit [http://localhost](http://localhost).

```sh
bash docker.sh
```

Run for PowerShell `docker.ps1`.

```ps1
& docker.ps1
```
